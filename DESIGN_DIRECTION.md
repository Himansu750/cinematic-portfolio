# Portfolio Design Direction

## Core Feeling

The portfolio should feel cinematic, premium, calm, and interactive. The site should look like a dark visual artist portfolio with Apple-level smoothness, not like a template or a marketing landing page.

## Visual Identity

- Dark matte black base, close to `#050505`.
- White text with controlled opacity instead of many colors.
- Soft glass/liquid surfaces for navigation and cards.
- Cinematic image/video content should be the main visual signal.
- Avoid noisy pixel effects, random gradients, heavy decorative blobs, or colorful UI that fights the footage.
- Keep the current rounded, liquid, premium surfaces, but make them clean and intentional.

## Typography

- Use Apple-style system typography.
- Large headings should feel cinematic and confident.
- Small labels should use uppercase tracking, but not so much that mobile becomes hard to read.
- Body text should be quiet, readable, and secondary.
- Avoid excessive negative tracking inside compact cards or buttons.

## Motion Rules

- Motion should feel smooth, responsive, and expensive.
- Use scroll reveals, soft blur fades, gentle 3D movement, and page transitions.
- Do not animate everything at once.
- 3D effects should help the user understand depth, not hide content.
- Final scroll states must always show the full content.
- Hover/tap feedback should be subtle and Apple-like.
- Respect reduced-motion settings.

## Layout Rules

- Desktop can be cinematic and spacious.
- Mobile and tablet should keep the same identity but use cleaner Apple/Pinterest-style layouts.
- No content should be cut off at the bottom or sides.
- No horizontal scroll.
- Navigation should never overlap important content.
- Cards should be easy to tap and visually readable.

## Page Priorities

1. Home page: first impression, portfolio intro, nav reveal, hero cards.
2. Visual Artist page: 3D tunnel stack reveal, smooth routing, clean final frame.
3. Category pages: cinematic project grids with finished-feeling layouts.
4. Footer/contact: simple, premium, usable.

## What To Avoid

- Random new colors.
- Generic SaaS/landing-page styling.
- Overly bright gradients.
- Too many boxes/cards inside cards.
- Heavy motion that makes the site feel fake or unstable.
- Redesigning away from the current cinematic identity.

## Working Method

Every design change should keep the existing style as the base. Improve one section at a time, verify mobile/tablet/desktop, then publish only after checks pass.
